## Usage

```bash
Usage: mdrs.exe [OPTIONS] <URL>

Arguments:
  <URL>  Folder or file to download

Options:
  -o, --output <OUTPUT>  Output directory [default: .]
  -m, --max <MAX>        Maximum number of concurrent downloads [default: 10]
  -h, --help             Print help
  -V, --version          Print version
```
